How to Run the Project 

1. Install Dependencies 
افتح الـ Terminal ونفذ الأمر التالي لتثبيت المكتبات ال
pip install pandas numpy matplotlib seaborn scikit-learn
2. Dataset Setup 
تأكد من وضع ملف البيانات marketing_campaign.csv في نفس المجلد (Folder) الذي يحتوي على ملف الكود.

3. Run the Code 
افتح ملف المشروع باستخدام Jupyter Notebook أو VS Code.

اضغط على Run All Cells لتشغيل الكود بالكامل.
